## How To Run The App:

In order to run the app locally through simulators or on a device these are the steps needs to be done

1. Clone this repo
2. Make sure we are in the **PetPalApp** folder
3. Run ```` npm start ```` from the terminal
4. Here you can decide to either download the Expo Go App and scan the QR Code
5. If running on a simulator, launch either an Andriod Simulator and press A into the terminal
